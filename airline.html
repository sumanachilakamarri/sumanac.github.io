Airline Management Systems

Contributors: Sumana Chilakamarri, Diya Mohan, Aditi Shastri, Minnie Chen

This project was part of my coursework at Georgia Tech through CS 4400: Intro to Database Systems.

Our task as a group was to analyze, specify, design, implement, and document an online system based on the provided scenario description through the use of querying in mySQL.

Project Phases

Phase I included to review the provided scenario description, and translate the requirements into an (Enhanced) Entity-Relationship Diagram (EERD)

Phase II included translating the EERD to a set of relational schema, identifying primary and foreign keys, and documenting unhandled constraints

Phase III included creating and implemented stored proceudres and views to provide information to system operators based on problem scenarios
